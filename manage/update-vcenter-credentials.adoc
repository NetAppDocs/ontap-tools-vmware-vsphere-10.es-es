= Actualizar las credenciales del servidor vCenter
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Puede actualizar las credenciales de la instancia de vCenter Server mediante la consola de mantenimiento.

.Antes de empezar
Debe tener credenciales de inicio de sesión de usuario de mantenimiento.

.Acerca de esta tarea
Si modificó las credenciales del servidor vCenter después de la implementación, actualícelas siguiendo este procedimiento.

.Pasos
. Desde vCenter Server, abra una consola al proveedor VASA.
. Inicie sesión como el usuario de mantenimiento.
. Acceda `2` a para seleccionar Menú de configuración del sistema.
. Ingresar `8` para cambiar las credenciales de vCenter.
