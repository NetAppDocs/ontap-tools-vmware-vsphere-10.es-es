= Restablecer contraseña de usuario de aplicación
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Siga estos pasos para restablecer la contraseña de usuario de la aplicación necesaria para el registro del proveedor SRA y VASA con vCenter Server mediante ONTAP tools for VMware vSphere.

.Pasos
. Abra un navegador web y navegue a: `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. Inicie sesión utilizando las credenciales de administrador configuradas durante la implementación de las herramientas ONTAP .
. Desde la barra lateral, seleccione *Configuración*.
. En la página *Credenciales VASA/SRA*, seleccione *Restablecer contraseña*.
. Ingrese y confirme la nueva contraseña.
. Seleccione *Restablecer* para aplicar la nueva contraseña.
