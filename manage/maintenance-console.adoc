= Conoce la consola de mantenimiento de las herramientas ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
La consola de mantenimiento de las ONTAP tools for VMware vSphere le permite gestionar la configuración de aplicaciones, sistemas y redes.  Puede actualizar las contraseñas de administrador y mantenimiento, generar paquetes de soporte, configurar los niveles de registro, administrar la configuración TLS y habilitar diagnósticos remotos.

Después de implementar las ONTAP tools for VMware vSphere, si la consola de mantenimiento no está accesible, instale las herramientas de VMware desde el servidor vCenter.  Inicia sesión usando el `maint` Nombre de usuario y contraseña establecidos durante la implementación.  Utilice *nano* para editar archivos en la consola de mantenimiento o de inicio de sesión de root.


NOTE: Debe establecer una contraseña para el `diag` usuario al activar el diagnóstico remoto.

Debe utilizar la pestaña *Resumen* de las herramientas de ONTAP implementadas para VMware vSphere para acceder a la consola de mantenimiento. Al seleccionar image:../media/launch-maintenance-console.gif["Botón de reproducción"], se inicia la consola de mantenimiento.
